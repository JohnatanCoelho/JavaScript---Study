## 📃 O que é JavaScript? 

JavaScript é uma linguagem de programação dinâmica e amplamente utilizada para adicionar interatividade a sites e aplicativos web. Com o advento de Node.js, também se tornou uma linguagem de uso comum no back-end, permitindo o desenvolvimento de aplicações completas com uma única linguagem!
